# Examples

This directory contains a few lightweight, public Riseholme examples for quick inspection.

## `images/`

Representative public sample imagery used for documentation and qualitative illustration.

## `outputs/`

Small example outputs generated by the mapping pipeline, including:

- detected or clustered pole GeoJSON
- topological-map exports
- qualitative inference visualisations

These examples are intentionally small and illustrative. They are not a substitute for the full datasets hosted on Zenodo.
